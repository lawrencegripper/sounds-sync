## Dev

Create a `.env` file with: 

```
APPLE_BEARER_TOKEN=
APPLE_USER_TOKEN=
APPLE_PLAYLIST=
SHOW_NAME=radio 1 chillest show
```

Get these by looking at the devtools of your browser while logged into `music.apple.com`.

Run `make run` and watch the songs get synced.
